# Last Frontier Voiceflow Styles

Custom CSS stylesheet for the Last Frontier Heliskiing Voiceflow chat agent.

## Usage in Voiceflow

Add this stylesheet URL to your Voiceflow widget configuration:

```
https://yannicksegaar.github.io/lastfrontier-voiceflow-styles/lastfrontier-theme.css
```

## Contents

- `lastfrontier-theme.css` - Main stylesheet with brand colors and typography
- `fonts/` - Nexa Rust Sans Black web fonts (WOFF2)

## Brand Colors

- Primary Red: `#cc3333` (Pantone 485)
- Black: `#000000`
- White: `#FFFFFF`
